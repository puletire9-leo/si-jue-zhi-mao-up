 三个问题逐个解决： 

  1. PowerShell 不支持 < 重定向

  # 错误：PowerShell 保留 < 给将来用
  docker exec -i ... < file.sql

  # 正确：用管道
  Get-Content file.sql | docker exec -i ...

  2. 密码被 Shell 环境变量覆盖

  你的 Shell 里 $env:MYSQL_ROOT_PASSWORD=root123456，compose 里 ${MYSQL_ROOT_PASSWORD:-root} 拿到的不是 root 而是        
  root123456。已改硬编码 root。

  3. dump 文件 41MB，PowerShell 管道卡死

  Get-Content -Raw 把 41MB 全读到内存再传，爆了。最终用 cmd /c type 流式管道：

  cmd /c "(echo USE sijuelishi; && type backend\database\backup\sijuelishi_full_20260515_160545.sql) | docker exec -i    
  sjzm-mysql mysql -uroot -proot -P3307"

  echo USE sijuelishi; 补了 dump 缺失的选库语句，type 逐行流式传输，不占内存。
